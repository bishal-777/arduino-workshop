# Arduino Workshop

This repository contains the basic Arduino example programs used during our Arduino Workshop.

The codes are designed for beginners and cover fundamental Arduino concepts.
## Requirements

- Arduino IDE
- Arduino Uno (or compatible board)
- Basic electronic components such as LEDs, resistors, push buttons, and sensors

## Purpose

The goal of this repository is to provide workshop participants with easy-to-understand examples that can be used for learning, practice, and experimentation with Arduino programming.

## Workshop

These examples were prepared for the **Arduino Workshop** conducted by the Advanced Robotics Club(ARC) with collaboration 
with Department of Computer and Electronics Engineering to introduce students to arduino programming and embedded systems.
